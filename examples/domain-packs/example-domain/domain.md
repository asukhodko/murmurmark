# Example Domain Pack

This is a safe public example of a MurmurMark domain pack.

Use it as a template for a private workspace-specific pack. Do not commit real
project names, people names, customer names, tickets, service names or internal
acronyms.

Typical public engineering vocabulary:

- reliability;
- observability;
- incident review process;
- deployment pipeline;
- latency;
- SLO/SLA/SLI;
- Kubernetes;
- Kafka.

Speech can mix Russian and English. Technical terms should usually keep their
canonical spelling.

Rules:

- do not translate technology names;
- keep common abbreviations in Latin characters;
- keep this tracked example generic;
- put private terms into ignored local files such as `glossary.yaml` or
  `whisper-prompt.ru.txt`.
